## How to Add These Tags

1. Go to the repository page on GitHub
2. Click on the **⚙️ Settings** icon (gear icon) in the **About** section (top-right of the repository page)
3. In the **Topics** field, add the relevant tags from the list above
4. Click **Save changes**
